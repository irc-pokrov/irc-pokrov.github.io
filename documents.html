Комплексна психолого-педагогічна оцінка розвитку дитини проводиться за попереднім записом.

 Документи, які потрібні для комплексної оцінки розвитку дитини:

  • Згода  батьків (одного з батьків) або законних представників на обробку персональних даних їх неповнолітньої дитини ( скачати форму у форматі word або pdf ) 
  • Заява щодо проведення комплексної психолого-педагогічної оцінки розвитку дитини ( скачати форму у форматі word та pdf ) 
  • Характеристика (завірена)
  • Форма первинної облікової документації № 112/о "Історія розвитку дитини" ( Медична карта Ф-112/о )
  • Копія свідоцтва про народження дитини
  • Паспорт для підтвердження особи батьків
  • Висновок в 2-х екземплярах
  • Зошити з рідної мови, математики
